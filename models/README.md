# Models

Detailed write-ups of the fixtures characterised in depth on hardware. For the capability
flags across all ~108 models in the family, see the
[model capability matrix](../model-capabilities.md).

## Detailed pages

The TL tube line shares a common protocol dialect and is the reference set for this
catalogue:

- [TL120C / TL120C-2](tl120c.md) — 120 cm RGB tube, pixel-capable, built-in battery
- [TL90C](tl90c.md) — 90 cm RGB tube, built-in battery (pixel vendor-listed, unverified)
- [TL60 RGB / RGB-2 / RGB-3](tl60.md) — 60 cm RGB tube; the RGB-2/RGB-3 add the Streamer feature
